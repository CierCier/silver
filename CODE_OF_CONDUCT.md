# Code of Conduct

## Our Standard

This project aims to be a respectful, technically serious, and human-centered space for collaboration.

We expect contributors and maintainers to:
- treat other people with respect
- assume good intent while still giving honest technical feedback
- communicate disagreement without hostility or contempt
- keep reviews, issues, and discussions focused on the work
- make room for people with different backgrounds and experience levels

## Unacceptable Behavior

Unacceptable behavior includes:
- harassment, intimidation, or personal attacks
- abusive or insulting language
- discrimination of any kind
- sustained bad-faith arguing, trolling, or deliberate disruption
- publishing private information without permission

## Enforcement

Project maintainers are responsible for setting and enforcing these standards.

They may edit, remove, or moderate comments, issues, pull requests, discussions, or other contributions that violate this code of conduct.

Repeated or serious violations may lead to temporary or permanent removal from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainers through the most appropriate available project channel.

Reports will be handled as carefully and respectfully as possible.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, code review, discussions, and other community interactions tied to this repository.
